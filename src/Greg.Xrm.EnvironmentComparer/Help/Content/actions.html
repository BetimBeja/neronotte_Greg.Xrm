Actions

The actions panel summarizes the list of actions that the user decided to apply after the comparison to streamline the deployment phase.
Actions are displayed (and applied) in the same order as they are defined.

User can choose to:

  • Clear the action list: all the actions are removed from the action list.
  • Remove a specific action from the list: removes the selected action from the list.
  • Apply all the actions: all the actions are applied in the same order they are defined.
  • Apply only the "checked" actions: all the actions that are checked in the list are applied, in the same order they are defined.
  • Select all the actions: all the actions are automatically checked.
  • Deselect all the actions: all the checks on the actions are automatically removed.

A confirmation prompt is displayed to the user before executing any specific operation.

If an action fails, a log is tracked in the Output view, and the system moves forward with the next one.

All the actions that have successfully been applied are automatically removed from the list.
At the end of the process, a new comparison is triggered in order to display the results of the applied actions.
