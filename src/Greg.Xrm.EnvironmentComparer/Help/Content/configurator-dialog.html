Configure table dialog

This panel allows to select an entity to compare between the two environments.

After choosing an entity you can also select which key should be used to detect whether a specific record is present in both environments or not.
You can choose to use as key for comparison the default GUID of the record, or to build a logical key picking one or more columns of the table.

The dialog allows also to configure which attributes should not be taken in consideration while comparing records, and whether or not consider only active records in the comparison.
