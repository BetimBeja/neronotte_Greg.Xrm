Configurator

The configurator tool allows to define the entities to compare.
Once defined the configuration can be saved to be reopened lately.

Prerequisites

To setup one or more entity configurations you should be connected to ENV1 and metadata from that environmnent should be loaded. To load table metadata you can click on the Actions > Load tables button.

Menu items

  • File
    • New: create a new comparison configuration
    • Open...: open a previously saved comparison configuration
    • Save: saves the current comparison configuration as a json file
    • Save as...: saves the current comparison configuration as a new json file
  • Actions
    • Load tables: loads table metadata from environment 1. It is mandatory to add or modify a table comparison configuration
    • Add table...: opens the "configure table" dialog that allows to configure how to compare a specific table
    • Edit...: opens the "configure table" dialog that allows to change the selected table comparison configuration
    • Remove: removes the selected table comparison configuration
  • Compare
    • Execute comparison: executes the comparison between the tables specified in the comparison

Please note: this tool is not suitable to compare large amount of data.
If you need to compare large amount of data we suggest to apply different approaches such as copying data to staging tables on Azure SQL and apply comparison via Stored Procedures.
