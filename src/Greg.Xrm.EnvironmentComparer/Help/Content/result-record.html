Record Info

The record info view allows to easily compare records that are present in both environments with mismatching values.
The record info view is automatically refreshed when the selected record on the Comparison Result Details grid changes.

A message at the top of the record info view describes the result for the selected record.
If the record is present in both environments with mismatching values, the mismatching values are shown side-by-side in the main area of the control.

The user can choose which attribute to compare using the Column selector at the top of the view.
The Column selector shows only the columns with mismatching values.

The user can use the two buttons on the top right of the window (Move to next record and Move to previous record) to change the comparison result currently displayed.
